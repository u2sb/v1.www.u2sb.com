## 介绍

可以插入 html5 原生 audio 标签

## 使用

```
{% mmedia "audio" "src:a.mp3" %}
```

```
{% mmedia "audio" "src:https://baidu.com/a.mp3" "autoplay:true" %}
```

## 参数

- 使用 `:` 或 `=` 分割。
- 所有 `<audio>` 标签的原生参数均可添加，只要能写进去就可以。
- 具体能否实现相关标准，取决于客户端浏览器。

## 配置

默认配置可写入 `_config.yml` 下

```
mmedia:
  audio:
    default:
      autoplay: true
      key: vaule
```
